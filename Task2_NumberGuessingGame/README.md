# Task 2: Number Guessing Game

## Description
A console-based Java game where the computer randomly generates a number between 1 and 100, and the user tries to guess it within a limited number of attempts. The program gives feedback on whether the guess is too high or too low, tracks scores across rounds, and supports multiple rounds of play.

## Features
- Random number generation (1–100)
- Limited attempts per round (10 attempts)
- Higher/lower feedback after each guess
- Input validation (handles invalid or out-of-range input)
- Scoring system based on number of attempts used
- Option to play multiple rounds with a running total score

## How to Run
1. Make sure you have Java (JDK 8+) installed.
2. Open a terminal in this folder.
3. Compile the program:
   ```
   javac NaziyaSyed_Task2.java
   ```
4. Run the program:
   ```
   java NaziyaSyed_Task2
   ```
5. Follow the on-screen prompts to guess the number.
